# 要件定義書: Claude チャット引き継ぎ拡張機能

## 1. 概要

Claude（ブラウザ版）のチャット内容を、新規チャット作成時に引き継ぐためのブラウザ拡張機能。
モデル変更（例: Sonnet → Haiku）などの際に、会話の文脈を損なわずに新規チャットへ移行することを目的とする。

---

## 2. 背景・目的

- Claude ブラウザ版では、新規チャットを開始すると過去の会話履歴が引き継がれない
- モデル変更時などに会話の文脈を継続したいニーズがある
- 手動引き継ぎを補助するツールとして、引き継ぎ用テキストを自動生成する

---

## 3. 機能要件

### 3.1 引き継ぎテキスト生成機能

| ID | 要件 |
|----|------|
| F-01 | 現在のチャット内容を解析し、処理経緯を要約したテキストを生成する |
| F-02 | 生成されたテキストはコピー可能な形式で出力する |
| F-03 | 引き継ぎテキストは新規チャットにそのまま貼り付けて利用できる形式とする |

### 3.2 データ受け渡し通知機能

| ID | 要件 |
|----|------|
| F-04 | 引き継ぎ時にファイル・データの受け渡しが必要な場合、その旨を明示するメッセージを表示する |
| F-05 | 必要なデータの種類や内容をメッセージ内に具体的に記載する |

### 3.3 手動引き継ぎサポート

| ID | 要件 |
|----|------|
| F-06 | 引き継ぎ作業そのもの（新規チャットへの貼り付け等）はユーザーが手動で行う |
| F-07 | 拡張機能はテキスト出力と通知のみを担い、自動転送は行わない |

---

## 4. 非機能要件

| ID | 要件 |
|----|------|
| NF-01 | Claude ブラウザ版（claude.ai）上で動作するブラウザ拡張機能として実装する |
| NF-02 | テキストデータのみを扱う（画像・ファイルの自動転送は対象外） |
| NF-03 | ユーザーの操作は最小限（ボタン1クリック程度）で完結する |

---

## 5. 対象外

- 自動的な新規チャットへの内容転送
- 画像・添付ファイルの自動引き継ぎ
- Claude API を使用したバックエンド処理

---

## 6. 想定ユーザーフロー

1. ユーザーが現在のチャット画面で拡張機能のボタンをクリック
2. 拡張機能が現在の会話を解析し、引き継ぎ用の要約テキストを生成
3. データ引き継ぎが必要な場合、その旨のメッセージを表示
4. ユーザーが生成されたテキストをコピー
5. ユーザーが新規チャットを開き、テキストを貼り付けて引き継ぎ完了

---

## 7. 成果物

- ブラウザ拡張機能（Chrome / Edge 向け Manifest V3 形式）
  - `manifest.json`
  - コンテンツスクリプト（チャット内容の取得・解析）
  - ポップアップ UI（引き継ぎテキスト表示・コピーボタン）
